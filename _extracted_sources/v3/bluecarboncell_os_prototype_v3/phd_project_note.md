# How to present BlueCarbonCell OS v3 in the PhD proposal

Use this wording:

A prototype digital twin, named BlueCarbonCell OS, will be developed as an early-stage decision-support tool for MCFC-based ship retrofit feasibility. The tool will translate MCFC cell/stack assumptions and ship exhaust data into ship-level indicators such as CO2 capture potential, heat recovery, pressure-loss risk, module size, thermal suitability, uncertainty range, pilot-readiness and retrofit feasibility score.

Do not call it a certified design tool. Present it as a transparent research prototype connecting experimental MCFC data, numerical modelling and ship-level retrofit decisions.
