
# BlueCarbonCell OS - Better UI Edition

This package improves the main UI while keeping **all previous code**.

- `app.py`

## Preserved previous code

All previous versions are kept:

- `pages/00_Previous_Main_v3_App_Preserved.py`

- `pages/02_Original_v2_Research_Dashboard.py`
- `pages/03_Original_v3_Advanced_MVP.py`
- `original_versions_do_not_delete/`

Nothing from the previous versions was removed.

## Run on Windows

Install requirements:

```bat
python -m pip install -r requirements.txt
```

Run:

```bat
python -m streamlit run app.py
```

Or use:

```bat
RUN_BLUECARBONCELL_OS.bat
```
